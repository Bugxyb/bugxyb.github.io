[A] Increasing Decreasing String

题目描述

给定字符串调整顺序，使其成为递增递减递增递减的新字符串

解题思路

模拟即可，按照所给的字符，从小到大再从大到小的排序给出即可

[B] Find the Longest Substring Containing Vowels in Even Counts

题目描述

找出最长的字串，使其aeiou这5个字符出现的次数是偶数次

解题思路

状态压缩的统计F[idx][n]表示到第idx位的时候，aeiou5个字符出现的次数分别是奇数还是偶数次的情况。
最后找出最大的F[idx][0]即可

代码

[C] Longest ZigZag Path in a Binary Tree

题目描述

给定一棵树，找到最长的一个树链，使其左右子树交替走，返回最长的长度

解题思路

遍历树即可，根据其父节点的情况，计算如果从父节点下来的时候，最大的长度

代码

[D] Maximum Sum BST in Binary Tree

题目描述

给定一棵树，找到一个子树，满足是BST，且这个子树所有结点的和是最大的

解题思路

如果一棵树是BST树，那么它的左子树和右子树都满足BST树的性质，遍历这棵树即可

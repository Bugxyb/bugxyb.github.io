[A] Count Negative Numbers in a Sorted Matrix

题目描述

解题思路

数据量小，可以O(nm)复杂度暴力解决；从右上角统计，可以做到O(n+m)的线性复杂度；

[B] Product of the Last K Numbers

题目描述

有add和getProduct操作，队列末尾加数和累乘最后K位数，返回计算结果

解题思路

因为有0的存在，以及只最后K位数的乘积，只需记录最后一个出现0的位置，以及之后累成的结果，getProduct可以做到O(1)返回

[C] Maximum Number of Events That Can Be Attended

题目描述

事件List，可以在[startDay, endDay]其中一天完成，每天最多完成一件事情，问最多完成多少件事情；

解题思路

贪心思路，每天完成endDay最近的时间，使用最大堆（优先队列）处理即可。

[D] Construct Target Array With Multiple Sums

题目描述

给你全是1的List，每次只能进行求和，然后替代其中的一个数，问你能不能达到目标对应的List

解题思路

倒推思想：每次求和之后的数都是List中最大的一个数，可以从最后目标List开始，选择最大的一个数，然后倒推，直到全是1为止

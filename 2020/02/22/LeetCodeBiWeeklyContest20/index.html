[A] Sort Integers by The Number of 1 Bits

题目描述

排序：先按数字2进制中1的位数从小到大排序，如果位数相同则按大小顺序排序。

解题思路

将数字进行二进制解析，解析后进行二维数组的排序即可

题目描述

给定N个商品和单价，然后依次客户进来购物，第N个客户打折Discount

解题思路

[C] Number of Substrings Containing All Three Characters

题目描述

给一个只包含abc的字符串，求字符串的字串同时包含abc的个数

解题思路

假设从第i个位开始到第j位已经包含了abc三个字符，那么从i位到j+1…length都包含abc字符，累加即可，O（n）

代码

[D] Count All Valid Pickup and Delivery Options

题目描述

有N个物品，需要拿起和放下操作。问一共有多少种方案进行拿起和放下

解题思路

数学题。假设已经放了N-1个物品了。那么操作第N个物品的时候，总共有2(N-1)+1个空位进行放置。
及fn(n) = fn(n-1) * (2(N-1)+1) * (2 * (N-1) + 1 + 1) / 2
